# Individual JSP Program

My individual JSP program for this project is a CRUD database for a place to record vehicle dynometer readings. The source code for this program can be found in my GitHub repository here.

After creating the dyno record database, you can add a new record using the create page.

Once you add all of the entries you want, you can view the current database entries by retrieving the records.

If there are any entires with incorrect information, you can edit them using "update dyno record".

The final process you can use for the database is to delete a record.
